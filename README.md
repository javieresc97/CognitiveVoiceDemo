# CognitiveVoiceDemo

Sample app to show Speaker Recognition API from Microsoft Cognitive Services using Xamarin.Forms.

## Before running it
You need to get your key from the [official site](https://azure.microsoft.com/en-us/services/cognitive-services/speaker-recognition/) and replace it [in the Keys file](https://github.com/javieresc97/CognitiveVoiceDemo/blob/master/VoicePay/Constants/Keys.cs).

## What's on it
- Basic speaker verification actions: create profile, enroll, get phrases, verify.
- Basic speaker identification actions: create profile, enroll, check enrollment status, identify, check identification status.
